# Pizza-Business-Analytics-Power-BI
# 🍕 Pizza Sales Analysis Dashboard

## 📊 Project Overview

This project focuses on analysing pizza sales data to uncover key business insights using **Power BI**. The dashboard provides a clear and interactive view of sales performance, customer behaviour, and product trends.

---

## 🚀 Key Features
* 📌 **KPI Metrics**

  * Average Order Value: **38.31**
  * Average Pizzas per Order: **2.32**
  * Total Orders: **21,350**
  * Total Pizzas Sold: **49,574**
  * Total Revenue: **817.86K**

* 📅 **Sales Trends**

  * Daily trend of total orders
  * Monthly trend analysis

* ⏰ **Busiest Days & Times**

  * Highest orders on **Weekends**
  * Peak days: **Friday & Saturday evenings**
  * Highest monthly sales: **July & January**

* 📦 **Sales Performance**

  * Category-wise contribution:

    * Classic, Supreme, Veggie, Chicken
  * Size-wise sales distribution:

    * Large, Medium, Regular, X-Large

* 📈 **Category Insights**

  * Classic category contributes the most to sales
  * Large-sized pizzas generate maximum revenue

---
